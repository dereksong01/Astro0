We’re going to show you the things Foundation can do!

Foundation provides this file (index.html) which includes some basic styles you can modify, play around with, or totally destroy to get going.

Foundation provides some grids that are very easy to manipulate! For example, this whole box is built using grids and cells. This top section, including the header and the two paragraphs is one cell that is 12 units wide. The following are divided into grids that are broken into cells that are 4 units wide each. Foundation provides some links to their pages. Below this section is Foundation's own demonstration of the grids.
